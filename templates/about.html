About This Project

Customer Churn Analysis using Machine Learning

Project Overview

Customer churn occurs when customers stop using a company's services. This project predicts customer churn using machine learning techniques and helps businesses improve customer retention.

Project Objective

The objective is to identify customers who are likely to leave and provide actionable business recommendations to reduce churn.

Technologies Used

  • Python
  • Flask
  • Pandas
  • NumPy
  • Scikit-Learn
  • HTML
  • CSS
  • Machine Learning

Machine Learning Model

Algorithm: Logistic Regression

Accuracy: 80.27%

Features: 5

  • Gender
  • Contract Type
  • Tenure
  • Monthly Charges
  • Internet Service

Project Workflow

  2. Data Cleaning
  3. Data Visualization
  4. Feature Selection
  5. Model Training
  6. Model Evaluation
  7. Flask Deployment

Business Benefits

  • Reduce customer loss.
  • Improve retention strategies.
  • Increase revenue.
  • Improve customer satisfaction.
  • Support business decisions.
